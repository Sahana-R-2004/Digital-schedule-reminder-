About Digital Schedule Reminder

Digital Schedule Reminder is a web-based application designed to help users efficiently manage their tasks and schedules. It allows users to add, view, and receive reminders for upcoming events or responsibilities, promoting better time management.

Key Features:
- User authentication and session management
- Add, view, and manage personal schedules
- Automated reminders using scheduling tools
- Responsive and user-friendly UI with a clean layout

This project was developed as a part of an academic initiative to apply practical skills in web development, user experience design, and time management automation.
